# demoos
**demoOS** is a **bare‑metal** operating system, designed for Raspberry Pi 3B, written in C and 
Assembly.  
The goal of this OS is to be accessible and easy to read, to help students learning what's inside 
an operating system. 

The core functionalities are:

- Multitasking
- Kernel and user space separation with system calls
- Virtual memory
- File system integration
- Inter process communication

Please visit [the official documentation](https://andrearanica.github.io/demoos/) to start using
demoos!
